# ‚ùì Business Questions Answered
The dashboard is structured to provide visual answers to the following 7 dimensions:

Revenue vs. Profit: Analyzes which product categories (e.g., Skincare) are driving the highest margins.

Transport Mode: Breaks down the distribution of shipping via Air, Rail, Road, and Sea.

Order Size: Identifies the most common order volumes to optimize inventory and packaging.

Customer Demographics: Segmenting buyers by gender (Male, Female, Non-binary, and Unknown) for targeted marketing.

Geography: Visualizing revenue distribution across key hubs like Mumbai, Delhi, and Kolkata.
